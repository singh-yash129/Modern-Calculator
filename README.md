# Modern Calculator

A sleek, modern calculator web application built with a responsive design. Perform basic calculations with ease in a visually appealing interface.

## Features

- Basic arithmetic operations (addition, subtraction, multiplication, division)
- Responsive design for desktop and mobile
- User-friendly interface

## Technologies Used

- **HTML** (48.8%)
- **JavaScript** (27.2%)
- **CSS** (24%)

## Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/singh-yash129/Modern-Calculator.git
   ```

2. **Open `index.html` in your browser**  
   No installation required. All files are static and contained in the repository.

## File Structure

- `index.html` — Main HTML structure of the calculator
- `style.css` — Styling and layout
- `script.js` — Calculator logic
